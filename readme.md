# えひめAI・IoT研究会技術セミナー資料


## 事前セットアップ
ご参加前に，このページ上部にある handson_install1.pdf を参照して，持参するノートパソコンへのセットアップを行ってください．

セットアップ中にもう一つのファイル keras_nn.py も使用します．ページ上部の keras_nn.py をクリックして keras_nn.py の内容（Pythonソースコード）が表示されたら，右上の Raw をクリックし，表示されたコードを keras_nn.py という名前でパソコンに保存してください．


## セミナー当日使用データ
セミナー当日の実習で使用するデータが handson_data.zip です．持参するノートパソコン上で予めダウンロードして保存しておいてください（ページ上部の handson_data.zip をクリックして，表示されたページの Download をクリックするとダウンロードできます）．


## 本データについて
本データは2019年4月6,7日開催のえひめAI・IoT研究会技術セミナーで使用するものです．
